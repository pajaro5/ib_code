## 🎯 Curriculum Coverage

These games cover the **IB CS SL 2027 syllabus**:

- **Topic A** — Computer fundamentals, Networks, Databases, Machine Learning
- **Topic B** — Computational Thinking, Programming in Python, OOP

Each game includes:
- Narrative hook + metacognitive pauses
- Immediate formative feedback (not just ✓/✗)
- Transfer question at the end
- Optional mini-game (retro mechanic) between levels

## 📚 Pedagogical Framework

Games are designed following:
- **Martin** (durable learning): progressive difficulty + consolidation moments
- **Furman** (meaningful learning): cognitive hook → conflict → exploration → feedback → transfer
